## Prerequisites
- Basic Python knowledge (variables, loops, functions)
- Linux command line familiarity
- Understanding of binary/hex numbers
- Network basics (TCP/IP)

## Lab Requirements
- Linux VM (Ubuntu 20.04+)
- Python 3.8+
- Kali Linux tools (optional)
- OWASP WebGoat or similar for practice

## Ethical Considerations
This course is for **educational and authorized security testing only**.
- Never attack systems without explicit authorization
- Understand legal implications (CFAA in US, Computer Misuse Act in UK)
- Follow responsible disclosure practices
- Use isolated lab environments only

---

Each module includes:
1. **Theoretical Background** - Understanding the "why"
2. **Technical Implementation** - Python code with detailed explanations
3. **Practical Exercises** - Hands-on labs
4. **Advanced Concepts** - Deep technical dive
5. **Defense Mechanisms** - How to protect against these techniques
