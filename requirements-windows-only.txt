# ============================================================
# TROUBLESHOOTING
# ============================================================

"""
Common Issues on Windows:

1. pip command not found:
   - Python not in PATH
   - Solution: Reinstall Python, check "Add Python to PATH"

2. Module import errors:
   - Packages not installed
   - Solution: Run pip install -r requirements-windows-only.txt

3. Permission denied errors:
   - Run PowerShell as Administrator
   - Or: Check file permissions

4. Scapy limited functionality:
   - Windows Scapy has reduced features
   - Solution: Use Wireshark for packet capture
   - Or: Use pyshark wrapper

5. pyodbc errors:
   - Microsoft ODBC driver not installed
   - Solution: Download ODBC driver or skip

6. Virtual environment activation issues:
   - ExecutionPolicy error
   - Solution: Run: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

For more help:
  - Python docs: https://docs.python.org/3/
  - Module docs: Check package PyPI pages
  - Stack Overflow: Search for "windows python [error]"
"""
